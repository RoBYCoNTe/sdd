# 001 - Task due date not saved

## Description

When editing a task and setting a due date, the date is not persisted after saving. Reopening the task shows no due date.

## Steps to reproduce

1. Create a new task
2. Set a due date
3. Save the task
4. Reopen the task — due date is empty

## Expected behavior

The due date should be saved and visible when reopening the task.
